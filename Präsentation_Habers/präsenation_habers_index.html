Node.js ist eine serverseitige Plattform mit, der sich Plattformen zum Betrieb von Netzwerkanwendungen realisieren lassen.

Node.js

  • Node.js ermöglicht es Serverseitig mit JavaScript zu arbeiten
  • Arbeitet wie alle JavaScript Anwendungen Eventbasiert
  • Arbeitet dadurch Ressourcen schonend
  • Läuft auf der "V8" Engine von Google, kann dadurch alles nutzen was Chrome kann
  • Bringt viele Pakete mit sich

Möglichkeiten

  • Mithilfe von Node.js lassen sich Webserver erstellen
  • Node.js arbeitet serverseitig mit JavaScript
  • Dadurch ist eine einfache Kommunikation zwischen Client und Server möglich

Code Möglichkeiten

  • "Child Processes"
  • "Cluster"
  • "HTTP"
  • "OS"
  • "Debugger"
  • "C/C++"
  • "File System"
